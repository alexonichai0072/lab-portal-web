# lab-portal-web

本仓库用于归档和发布多个独立 HTML 页面。这些页面通常作为实验性、临时性或辅助性的前端内容，不针对任何单个域名或具体网站，方便统一管理与分发。

## 项目简介

`lab-portal-web` 是一个轻量级的静态页面集合仓库。它旨在汇集各类独立的 HTML 文件，每个文件可视为一个独立的“实验页面”或“工具页面”。仓库不依赖特定的后端服务，所有页面均为纯前端实现，可直接通过浏览器打开或部署至静态托管服务。

## 目录说明

仓库根目录下存放各 HTML 文件及其相关资源（如 CSS、JavaScript、图片等）。建议按功能或用途对文件进行命名，以便快速识别。例如：

- `index.html`：入口页面（可选）
- `tool-xxx.html`：工具类页面
- `experiment-yyy.html`：实验性页面
- `archive-zzz.html`：归档或历史页面

每个页面应尽可能自包含，减少跨文件依赖。如有共用样式或脚本，可放置在 `assets/` 或 `common/` 目录下。

## 页面归档说明

本仓库的 HTML 页面可能包含以下类型：

- 临时测试页面
- 功能原型或概念验证
- 离线可用的工具页面
- 不再维护但需保留的历史版本

所有页面均以静态方式归档，不保证持续更新或兼容所有浏览器。使用时请根据实际情况自行评估。

## 维护说明

### 贡献方式

欢迎提交 Pull Request 添加新的 HTML 页面或修正现有内容。请确保：

- 页面功能独立，不依赖外部不可控资源
- 文件命名清晰，避免特殊字符
- 如有必要，在页面内添加简短的说明注释

### 使用建议

- 可直接将仓库克隆到本地，通过浏览器打开 HTML 文件
- 也可将整个仓库部署到任何静态托管服务（如 GitHub Pages）进行访问
- 不推荐在敏感或生产环境中直接使用未经验证的页面

### 注意事项

- 本仓库不收集任何用户数据
- 页面内容不涉及商业推广或诱导行为
- 如发现页面存在安全或功能问题，欢迎提交 Issue
